
1. Проверяем наступление события, 
приводящие к построению/изменению графика:
	1.1. Создание графика
	1.2. Просрочка
	1.3. Частичное досрочное погашение
	1.4. Полное досрочное погашение
	1.5. Изменение лимита
	1.6. Кредитные каникулы
	1.7. Изменение процентной ставки
	1.8. Пролонгация
	1.9. Льготный период
	....................................
2. Если изменилась продолжительность графика,
генерируем (начиния с даты события) списки
	- дат погашения ОД
	- дат погашения %
3. Начиния с даты события,
строим покрытие интервалами
редуцированных расчетов ОД и %
4. Рассчитываем выплаты
	- по ОД
	- по %
Сохраняем посчитанные графики внутри события.
5. Если текущая дата < даты окончания договора, -> {1},
   если	текущая дата >= окончания договора, - закрываем график.
